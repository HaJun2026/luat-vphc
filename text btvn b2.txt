  ---
  Bước tiếp theo để có URL public

  1. Tạo GitHub repo và push code

  # Mở terminal tại C:\Claude code\luat-vphc
  cd "C:\Claude code\luat-vphc"
  git init
  git add .
  git commit -m "Initial commit: full-stack law violation lookup"

  Sau đó tạo repo trên GitHub (tên: luat-vphc) rồi:
  git remote add origin https://github.com/<username>/luat-vphc.git
  git push -u origin main

  2. Deploy trên Railway

  A. Backend + Database:
  1. Vào https://railway.app → New Project → Deploy from GitHub repo
  2. Chọn folder backend (hoặc config Root Directory = backend)
  3. Add Plugin → PostgreSQL (Railway tự tạo DATABASE_URL)
  4. Railway sẽ auto-detect Python và chạy Procfile
  5. Sau khi deploy xong → vào Shell của service → chạy python seed.py
  6. Copy URL backend (dạng xxx.railway.app)

  B. Frontend:
  1. Add Service mới → cùng GitHub repo → Root Directory = frontend
  2. Add Variable: NEXT_PUBLIC_API_URL=https://<backend-url>.railway.app
  3. Deploy → Railway tự chạy npm run build && npm start
  4. Copy URL frontend → đây là link public cho mọi người
